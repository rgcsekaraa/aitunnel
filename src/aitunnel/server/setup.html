Paste your Gemini cookies

Both __Secure-1PSID and __Secure-1PSIDTS are required for the first bootstrap. After that, the server auto-rotates PSIDTS indefinitely - you won't need to re-paste until PSID expires (months).

⚠ __Secure-1PSIDTS rotates ~hourly. Copy it now from a fresh gemini.google.com tab and submit within a few minutes.

How to find them
  1. Open https://gemini.google.com - you must see the chat UI, not a login page.
  2. Press F12 (or ⌥⌘I on Mac).
  3. Application tab → Cookies → https://gemini.google.com.
  4. Copy the Value of __Secure-1PSID and __Secure-1PSIDTS.
  5. Paste below and click Save.
